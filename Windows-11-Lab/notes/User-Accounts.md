# Local User Accounts Lab

## Objective

Practice creating and managing local Windows user accounts.

## Steps Completed

- Opened Computer Management
- Navigated to Local Users and Groups
- Created a new user named JohnSupport
- Configured password settings
- Verified the account was created

## Why This Matters

Help Desk technicians frequently create user accounts for new employees, reset passwords, unlock accounts, and manage account permissions.

## What I Learned

I learned how to create and manage local Windows accounts using Computer Management.
