# Event Viewer Lab

## Objective

Learn how to use Event Viewer to investigate Windows issues.

## Tool Used

* Event Viewer

## Findings

**Log Name:** System

**Source:** DistributedCOM

**Event ID:** 10010

**Level:** Error

## Description

The server did not register with DCOM within the required timeout.

## What I Learned

I learned how to locate Windows system errors using Event Viewer.

I learned how to identify the Event ID, Source, Level, and Description of an event.

I also learned that not every Event Viewer error means Windows is broken. Technicians use context before deciding whether an issue needs further troubleshooting.
